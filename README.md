# qr19: The Precision QR Engine
### Distilled. Jargon-free. Professional.

The world does not need another generic QR generator. It needs a better one. One that respects user privacy, avoids parasitic advertising, and delivers high-fidelity output for professional workflows.

Welcome to qr19. The precision engine for users who care about pixels.

## The Streamlined Workflow
We have maximized productivity by eliminating redundant steps.
1. Choose Content: Select the required QR type (URL, vCard, WhatsApp, etc.).
2. Design & Export: Fine-tune the aesthetic and download the result. 

## Technical Capabilities
We support a range of professional content protocols:
- Website URLs: Direct links to any web resource.
- Contacts (vCard): Seamless digital business card sharing.
- Communication: Direct WhatsApp and SMS links without manual contact saving.
- Email: Pre-filled subject and body parameters.
- Calendar Events: Invitation data for standardized scheduling.

## Studio Aesthetics
qr19 moves beyond default styling to achieve high-end SaaS professionalism:
- Studio Presets: Curated palettes including Onyx, Midnight, and Electric for instant visual consistency.
- Precision Branding: Integrated logo support with optimized excavation logic to ensure scannability.
- Vector & Raster Export: High-resolution PNG for standard use and SVG for vector precision.

## Zero-Noise Policy
- No Jargon: Technical terminology is simplified for human clarity.
- No Tracking: We do not collect or store user data.
- No Advertising: A clean, focused workspace free from third-party distractions.

## Technical Pedigree
Built with React, Vite, and Tailwind CSS. Engineered for stability and performance.

## Installation
```bash
# Clone the repository
git clone https://github.com/prasanna192005/qrgen.git

# Install dependencies
npm install

# Run the development server
npm run dev
```
